## About me
Full Stack Developer, specializing in creating intuitive and efficient web interfaces. I work daily with technologies like React.js, Next.js, and Node.js, and manage projects in agile environments using tools such as Jira and Kanban boards. I am characterized by a problem-solving mindset and a focus on optimized solutions, approaching each issue from different angles to select the most effective option. My passion for design enables me to translate prototypes into precise visual interfaces and propose improvements to enhance user experience.

I have contributed to the development of platforms with advanced functionalities for content management, API interaction, and data visualization. My experience spans from planning to implementation and continuous improvement of solutions, always geared towards scalability and user-friendliness.
